# wowlab

## Commands

```bash
pnpm build        # Build all (always use this)
pnpm dev          # Dev mode
pnpm test         # Run tests
pnpm lint         # Lint
pnpm cli <cmd>    # CLI tools
```

Never use `pnpm --filter` or `pnpm typecheck`.
