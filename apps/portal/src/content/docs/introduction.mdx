# Introduction

Welcome to WowLab. This documentation uses academic-style citations to reference external sources.

## Citation Examples

Inline citations appear as superscript links. For example, the WebSocket protocol <Cite id="websocket" /> enables full-duplex communication between clients and servers.

You can cite multiple sources in sequence. Our architecture uses Redis <Cite id="redis2024" /> for coordination and Centrifugo <Cite id="centrifugo2024" /> for real-time messaging.

Authentication relies on Ed25519 signatures <Cite id="ed25519" /> for cryptographic security, while persistent data is stored in Supabase <Cite id="supabase2024" />.

## How Citations Work

2. Use `<Cite id="key" />` inline where needed
