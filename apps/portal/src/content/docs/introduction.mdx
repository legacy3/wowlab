# Introduction

Welcome to WowLab. This documentation uses academic-style citations to reference external sources.

## Citation Examples

Inline citations appear as superscript links. For example, the <Cite id="websocket">WebSocket protocol</Cite> enables full-duplex communication between clients and servers.

You can cite multiple sources in sequence. Our architecture uses <Cite id="redis2024">Redis</Cite> for coordination and <Cite id="centrifugo2024">Centrifugo</Cite> for real-time messaging.

Authentication relies on <Cite id="ed25519">Ed25519 signatures</Cite> for cryptographic security, while persistent data is stored in <Cite id="supabase2024">Supabase</Cite>.
