# MCP Server

Query WoW spell and item data using AI assistants via MCP (Model Context Protocol).

## What it does

- Look up spells and items by ID or name
- Query raw DBC tables with filters
- Compute damage, durations, cooldowns via extractor functions
- Works with Claude Code, Cursor, and other MCP-compatible tools

## Install

```bash
npx @wowlab/mcp-server@latest
```
